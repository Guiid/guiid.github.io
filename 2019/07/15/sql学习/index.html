sql学习

Posted on 2019-07-15

sql学习：

sql是关系型数据库，RDBMS是关系型数据库管理系统，是所有关系型数据库的基础；

数据库表：

RDMBS中的数据储存在被成为表的数据对象中，数据包由行和列组成。

字段

每张表都可以划分出很小的一块数据，这块数据被称为字段，由行与列的坐标组成；

NULL

null值是以空白形式出现的值，表示该记录在此字段处没有设值；

SQL约束

DEFAULT 约束：提供该列数据未指定时所采用的默认值

NOT NULL：保证列中没有NULL值；

UNIQUE：保证列中的所有数据各不相同

主键：唯一标识数据表中的行记录

外键：唯一标识其他表中的一行记录

CHECK约束：此约束保证列中的所有值满足某一条件

索引：用于在数据库中快速创建与检索数据；

主键是每张表中的唯一一条记录的子u但，主键必须要包含唯一值，且不能是null；

每张表只有一个主键，一个组建可以有多个字段组成，成为组合键；

SQL分类

DML与DDL

DML数据操作语言：

  • SELECT 从数据表中获取数据
  • UPDATE 更新数据表中的数据
  • DELETE 从数据库中删除数据
  • INSERT INTO 向数据表中出入数据

DDL数据定义语言：

  • CREATE DATABASE 创建数据库
  • ALTER DATABASE 修改数据库
  • CREATE TABLE 创建新表
  • ALTER TABLE 更改数据库
  • DROP TABLE 删除表
  • CREATE INDEX 创建索引
  • DROP INDEX 删除索引
